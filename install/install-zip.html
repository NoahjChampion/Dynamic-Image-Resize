--- layout: layout title: Upload ZIP via FTP ---
  1. Download the Zipball from GitHub.
  2. Unzip it locally. I personally recommend using 7zip, but every other ZIP program will do it as well.
  3. Upload it via (S)FTP to your plugins or MU-plugins folder.
  4. Go to your admin UI.
  5. Navigte to the "Plugins" list.
  6. Activate it.
