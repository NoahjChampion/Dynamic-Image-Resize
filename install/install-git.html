--- layout: layout title: Install using Git ---

Open Git Bash or your Command line tool of choice.

$ git clone --recursive git@github.com:franz-josef-kaiser/Dynamic-Image-Resize.git folder-to-clone-into

Git Submodule

Please keep in mind that you need to check back for version updates on a regular base.

{% highlight console %} $ git submodule add git@github.com:franz-josef-kaiser/Dynamic-Image-Resize.git folder-to-clone-into # List our submodules $ git submodule -v # List our submodules from the gitmodules file $ cat .gitmodules {% endhighlight %}
