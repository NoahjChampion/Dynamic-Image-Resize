There are two possible use cases: Either place the shortcode inside your post content, or use a template tag.

Shortcode

Place [dynamic_image] in your content. The shortcode has four arguments:

  • src Full path to the image in your upload directory or the ID
  • width Integer value
  • height Integer value
  • classes Css classes â€“ separated by a -whitespace

Template Tag

The class uses the __toString() method. This method - as most magical PHP methods - only trigger under a certain condition. In this case, when you echo the output.

Inside the loop:

{% highlight php linenos %} echo dynamic_image_resize( array( 'src' => array_shift( wp_get_attachment_image_src( get_post_thumbnail_id( get_the_ID() ), "Full" ) ), 'width' => 60, 'height' => 100, 'classes' => 'thumb', ) ); {% endhighlight %}

Outside the loop with a path to an attachment:

{% highlight php linenos %} // The args need to be an array echo dynamic_image_resize( array( // The full path to the image in your uploads folder 'src' => 'http://example.com/wp-content/uploads/2012/03/some_image.png', 'width' => 60, 'height' => 100, 'classes' => 'some classes to align and style the image', ) ); {% endhighlight %}

Outside the loop with an attachment ID:

{% highlight php linenos %} // The args need to be an array echo dynamic_image_resize( array( // the ID 'src' => 6, 'width' => 60, 'height' => 100, 'classes' => 'some classes to align and style the image', ) ); {% endhighlight %}
